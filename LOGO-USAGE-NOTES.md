# Shellz provider logo notes

This package adds local provider logo tiles so review, coupon, comparison, and homepage cards are easier to scan.

Important rules:

- Provider logos are shown for identification and comparison only.
- Do not describe Shellz as an official partner unless the affiliate program explicitly permits that wording.
- Shellz structured data should use only the Shellz logo as publisher.logo.
- Replace placeholder/local wordmark tiles with official affiliate-dashboard or press-kit logo assets when available.
- Keep affiliate image links marked with rel="sponsored nofollow" when the image itself links to a provider checkout page.
